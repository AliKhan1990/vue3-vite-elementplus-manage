- [Es6+](http://es6.ruanyifeng.com/) - Familiar with es6 basic syntax
- [Vue-Router-Next](https://next.router.vuejs.org/) - Familiar with the basic use of vue-router

# 快速开始

推荐使用 pnpm

```shell
pnpm install
```

运行命令

```shell
pnpm run dev
```
